# 斗鱼弹幕监控助手

> ⚠️ **该脚本全部由AI编写**
>
> 一个 Tampermonkey 油猴脚本，用于监控斗鱼直播间弹幕，自动跟随热门弹幕发送。

## 功能特性

- 🔥 **热门弹幕检测**：实时监控5秒内出现次数最多的弹幕，达到阈值自动发送
- 📊 **TOP5 排行**：实时展示5秒内出现次数最多的5条弹幕及其计数，每秒更新
- 🚫 **屏蔽词过滤**：内置默认屏蔽词，支持自定义添加/删除，过滤进场消息、礼物消息
- 📜 **发送记录**：记录每次自动发送的弹幕内容、时间和触发计数
- 🔄 **智能去重**：已发送的弹幕1小时内不会重复发送
- 🖱️ **可拖拽面板**：悬浮窗支持拖拽移动，可折叠/展开
- 💾 **本地存储**：屏蔽词、触发阈值、面板位置等配置自动保存，刷新不丢失
- ⚙️ **阈值可调**：触发发送的弹幕次数阈值可在面板中直接编辑

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 Tampermonkey 图标 → **添加新脚本**
3. 删除默认内容，将 `douyu-danmu-monitor.user.js` 的代码粘贴进去
4. 保存（Ctrl+S）

## 使用方法

1. 打开任意斗鱼直播间页面
2. 页面右上角会出现悬浮窗控制面板
3. 脚本自动运行，无需手动操作

### 控制面板说明

| 区域 | 说明 |
|------|------|
| 状态栏 | 显示运行状态、监控弹幕数、已发送数 |
| 🔥 5秒热门弹幕 TOP5 | 实时显示当前5秒内出现次数最多的5条弹幕 |
| 📊 触发阈值 | 可编辑的数字输入框，修改后立即生效 |
| 🚫 屏蔽词 | 点击「管理」展开，支持添加/删除/清空屏蔽词 |
| ✅ 最新发送 | 显示最近一次自动发送的弹幕内容 |
| 📜 发送记录 | 点击「展开」查看历史发送记录 |

### 操作方式

- **拖拽移动**：按住面板顶部标题栏拖拽
- **折叠面板**：点击右上角「−」按钮，面板会滑出屏幕，边缘显示小图标
- **恢复面板**：点击边缘的 🎯 图标
- **暂停/恢复**：点击「停止/开始」按钮

## 配置项

| 配置 | 默认值 | 说明 |
|------|--------|------|
| 时间窗口 | 5秒 | 统计弹幕频率的时间范围 |
| 触发阈值 | 30条 | 5秒内相同弹幕达到此次数则自动发送 |
| 发送冷却 | 3秒 | 两次自动发送之间的最小间隔 |
| 去重窗口 | 1小时 | 已发送弹幕的重复发送冷却时间 |
| 最大记录 | 50条 | 发送记录最多保存条数 |
| TOP5显示 | 5条 | 热门弹幕排行显示条数 |

## 默认屏蔽词

有一些默认屏蔽词，但是这个爆了就不写了

> 屏蔽词采用**包含匹配**，弹幕中只要包含屏蔽词就会被过滤。可在面板中自由管理。

## 过滤机制

脚本会自动过滤以下类型的消息：

| 类型 | 过滤方式 |
|------|----------|
| 进场消息 | DOM类名检测 + 关键词检测 |
| 礼物消息 | DOM类名检测 + 关键词检测 + 礼物图标检测 |
| 系统消息 | 关键词检测 |
| 屏蔽词弹幕 | 包含匹配 |

## 本地存储

以下配置会自动保存到浏览器 `localStorage`，刷新页面后自动恢复：

- 屏蔽词列表
- 触发阈值
- 面板位置

> 清除浏览器缓存会丢失保存的配置。

## 兼容性

- 浏览器：Chrome / Edge / Firefox（需安装 Tampermonkey）
- 页面：斗鱼直播间（房间号页面、beta页面、topic页面）
- 不在主页、鱼吧等非直播间页面运行
